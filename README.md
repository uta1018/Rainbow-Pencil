# 虹色えんぴつ


## 概要
北海道大学集中講義「プログラミング構築手法を学び、論理的思考を習得しよう」(2023年度1学期開講)で開発したイラスト描画アプリです。

子どものころに好きだった、真っ黒な紙を鉛筆で削ると、虹色の線が引けるお絵描き帳を再現しました。

オーディエンス投票で50%を超える投票をいただきました。
